# Installation

## Prerequisites
- Python 3.8 or higher
- pip package manager
- Git (for development)

## Install via pip
```bash
pip install scrapeuncle
```

## Verify Installation
```bash
scrapeuncle --version
```

## Development Installation
If you want to install from source:

```bash
git clone https://github.com/scrapuncle-tech/scrapeuncle.git
cd scrapeuncle
pip install -e .
```
